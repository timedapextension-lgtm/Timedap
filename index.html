How it works
1) Add the Chrome extension

Tracks active tabs and saves website time locally. No account, no cloud.

2) (Optional) Install the desktop app

Monitors the active application to include native apps. Data is stored in a local tracking_data.json.

3) Review & export

Open the report, filter by tags or dates, and export to CSV, Excel, or PDF.

Transparency

Permissions

We request only what’s needed for tracking and icons.

  • activeTab & tabs for current site / tab time
  • storage to save logs locally
  • Network access only to fetch favicons from Google’s favicon service

Icons (Favicons)

Site icons are fetched via Google’s favicon service using the domain only. Activity data is never transmitted.

Example: https://www.google.com/s2/favicons?domain=<domain>&sz=32

Your data, your device

Delete or export at any time. Uninstalling removes the extension data; the desktop JSON file remains on your machine until you delete it.
